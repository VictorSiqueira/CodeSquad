  • ng-model: Serve para declarar os modos que vao ser usados
  • ng-controller: Serve para especificar que dentro do elemento que está marcado, haverão uma série de comandos
  • ng-bind: è utilizado no lugar das expressões que no meio vai o nome do model, {{model}}
  • ng-class: Serve para aplicar classes css
  • ng-show: Serve para mostrar ou não o elemento marcado
  • ng-hide: Serve para esconder ou não o elemento marcado
  • ng-repeat: serve para criar um loop
